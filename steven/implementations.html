ABC Implementations

The available files are:

NEW Windows/DOS
for the PC version. This is a new version with much better memory handling than the previous version. Included .pif files for running abc on Windows 98, Windows 2000 and XP. To install it, just unpack the ABC directory as c:\Program Files\ABC, or read the file readme.txt for what to do if you want it somewhere else. To uninstall, just delete the directory where you unpacked it.
abc-unix.tar.Z
for the Unix version (These are sources: also compilable on other machines if you have the right environment, such as Cygwin for the PC)
NEW abc-redhat-linux.tar
Binaries compiled under Redhat 7.3. See the readme for where to put the files.
abc.mac.sit.hqx
for the Mac version; not tested on newer Macs
abc.intro
for an overview of ABC (also included with the implementations)
abcst.arc
for the Atari ST version, for historical interest...
